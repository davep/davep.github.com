Yet another older [Emacs Lisp](/tag/emacs-lisp/) package that has had a tidy
up. This one is [`slstats.el`](https://github.com/davep/slstats.el), a wee
package that can be used to look up various statistics about the Second Life
grid. It's mainly a wrapper around the API provided by [the Second Life grid
survey](https://gridsurvey.com/).

When `slstats` is run, you get an overview of all of the information
available.

![An overview of the grid](/attachments/2026/04/15/sl-overview.webp#centre)

There are also various commands for viewing individual details about the
grid in the echo area:

- `slstats-signups` - Display the Second Life sign-up count
- `slstats-exchange-rate` - Display the L$ -> $ exchange rate
- `slstats-inworld` - Display how many avatars are in-world in Second Life
- `slstats-concurrency` - Display the latest-known concurrency stats for
  Second Life
- `slstats-grid-size` - Display the grid size data for Second Life

There is also `slstats-region-info` which will show information and the
object and terrain maps for a specific region.

![Region information for Da Boom](/attachments/2026/04/15/slstats-region-info.webp#centre)

As with a good few of my older packages: it's probably not *that* useful,
but at the same time it was educational to write it to start with, and it
can be an amusement from time to time.
