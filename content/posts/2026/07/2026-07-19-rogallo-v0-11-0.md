A quick update to [Rogallo](https://rogallo.davep.dev/), with changes
concentrating on giving the user more control over the content of a rendered
page.

Despite the fact that the [Gemini Protocol](/tag/gemini-protocol/) and
[gemtext](/tag/gemtext/) are all about things being minimal, some capsules
can deliver pretty busy content. Personally, I rather like it, and I'm
generally impressed at how creative some people are with these tools. On the
other hand, I've noticed that some people don't always appreciate the use of
emojis or ANSI escape sequences to provide colour. So, with this in mind,
I've added a couple of features to help control the content.

The first is [a command to toggle emoji
removal](https://rogallo.davep.dev/configuration/#emoji-removal). If you're
finding the emojis in a page too distracting, use the command to toggle them
off and on (bound to <kbd>F6</kbd> by default).

![AstroBotany without emojis](/attachments/2026/07/19/emoji-off.webp#centre)

The setting is sticky too, so if you're so inclined, you can turn emojis off
and they'll stay off.

There's a similar command related to the handling of ANSI escape sequences.
While some applications let you toggle this at the capsule level
(AstroBotany is one example where it's part of the game's settings), not all
do. So I've added a command to toggle this (bound to
<kbd>Shift</kbd>+<kbd>F6</kbd> by default).

![ANSI colour off](/attachments/2026/07/19/ansi-off.webp#centre)

With both these commands, if this sort of view isn't to your taste:

![AstroBotany and its rich display](/attachments/2026/07/19/full-display.webp#centre)

You can quickly and easily turn it into this:

![Colour and emojis off](/attachments/2026/07/19/all-off.webp#centre)

I think I prefer the richer view, but I can see the utility of the more
austere view.

Somewhat related to the above, I've also added the ability to configure the
main "icons" that get used inside the rendered content. So if you don't like
the choice of `⪢` for links within a capsule, `↗` for links outwith a
capsule, or `•` for a list item bullet, [you can change them in the
configuration file](https://rogallo.davep.dev/configuration/#icons).
