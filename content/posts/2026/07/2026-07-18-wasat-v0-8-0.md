[Wasat](https://wasat.davep.dev/)
[v0.8.0](https://wasat.davep.dev/changelog/#v080) is now available. This
provides a small update to the `GeminiURI` class, adding three utility
properties for extra URI manipulation.

- `without_query` - gives the URI minus any attached query
- `parent` - gives the parent path of the current URI (also minus the query)
- `root` - gives the root path for the current URI (also minus the query)

In each case, the result is a fresh `GeminiURI` instance.

These are added because I'm aiming to add "go to parent" and "go to root"
commands to [Rogallo](https://rogallo.davep.dev/).
