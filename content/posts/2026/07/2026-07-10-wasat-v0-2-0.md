I've made a quick update to [Wasat](https://wasat.davep.dev/), my async
[Gemini Protocol](/tag/gemini-protocol/) client library for
[Python](/category/python/). Now that I'm at a point where I want to add
client certificates to [Rogallo](/tag/rogallo/), I'm essentially shaking
down the support for this in Wasat.

One thing I wanted right away was a certificate that, in effect, never
expires. So with this release of Wasat I've added an option where the days
to expire can be set to `None`, which results in the expiration date being
set to 9999-12-31.
