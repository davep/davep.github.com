It's now well over a year since I [released
Braindrop](/2025/01/03/braindrop.html) and it's in constant use by me. I
continue to find [`raindrop.io`](https://raindrop.io/) a really useful
resource, and more often than not manage, edit, tag, and review what I save
there with Braindrop, including which [become
public](https://raindrop.io/davep/public-46742255/sort=-created&perpage=30&page=0),
and which don't.

I've made [a few small changes to the application in the past year and a
bit](https://github.com/davep/braindrop/releases), but not much. It's been
stable and useful. But on the back of a [recent change I made to
OldNews](/2026/04/15/oldnews-v1-4-0.html), I felt I needed to make the same
change here.

So with the release of v1.1.0 I've added three new commands to the
application:

- `JumpToNavigation` - Jump to the navigation panel; bound to <kbd>1</kbd>
  by default
- `JumpToRaindrops` - Jump to the main raindrops list panel; bound to
  <kbd>2</kbd> by default
- `JumpToDetails` - Jump to the details panel for the selected raindrop, if
  the panel is visible; bound to <kbd>3</kbd> by default

Now it's just a little easier and quicker to get around the UI.

If `raindrop.io` is your thing, and you want to work with your saved bookmarks
in the terminal: Braindrop is licensed GPL-3.0 and available [via
GitHub](https://github.com/davep/braindrop) and also [via
PyPI](https://pypi.org/project/braindrop/). It can also be installed using
[`uv`](https://docs.astral.sh/uv/getting-started/installation/):

```sh
uv tool install braindrop
```

If you don't have `uv` installed you can use [uvx.sh](https://uvx.sh) to
perform the installation. For GNU/Linux or macOS or similar:

```sh
curl -LsSf uvx.sh/braindrop/install.sh | sh
```

or on Windows:

```sh
powershell -ExecutionPolicy ByPass -c "irm https://uvx.sh/braindrop/install.ps1 | iex"
```

If `uv` isn't your thing then it can also be installed with `pipx`:

```sh
pipx install braindrop
```

Once installed, run the `braindrop` command.
