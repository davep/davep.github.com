## Introduction

If you visit [my GitHub profile](https://github.com/davep) you'll find a
*lot* of repositories. I have projects big and small, spanning many years
(some of them pre-dating the existence of GitHub by a long way). Here I'll
highlight some of the projects I'm actively using and maintaining. That
doesn't mean the others aren't used, or are neglected, it's just these are
the ones that have my main focus at the moment and I suspect others might
appreciate.

## BlogMore

![BlogMore](/attachments/active-projects/blogmore-social-banner.webp#centre)

[BlogMore](https://blogmore.davep.dev/) is a blog-oriented static site
generator. It's the tool I use to create this blog.

## blogmore.el

[blogmore.el](https://github.com/davep/blogmore.el) is an Emacs package that
serves as a companion to [BlogMore](https://blogmore.davep.dev/). I use it
heavily when writing, managing and updating posts for this blog.

## Braindrop

![Braindrop](/attachments/active-projects/braindrop-social-banner.webp#centre)

[Braindrop](https://github.com/davep/braindrop) is a terminal-based TUI
application for working with bookmark data stored in
[raindrop.io](https://raindrop.io/).

## Complexitty

![Complexitty](/attachments/active-projects/complexitty-social-banner.webp#centre)

[Complexitty](https://complexitty.davep.dev/) is a terminal-based TUI for
playing with and exploring Mandelbrot sets.

## Hike

![Hike](/attachments/active-projects/hike-social-banner.webp#centre)

[Hike](https://hike.davep.dev/) is a terminal-based TUI browser for Markdown
files.

## OldNews

![OldNews](/attachments/active-projects/oldnews-social-banner.webp#centre)

[OldNews](https://oldnews.davep.dev/) is a terminal-based TUI client for
[TheOldReader](https://www.theoldreader.com/en/). It's one of my
daily-drivers when it comes to reading RSS-based feeds.
